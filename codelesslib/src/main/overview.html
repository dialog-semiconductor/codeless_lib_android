The CodeLess™ library implements the Renesas CodeLess™ and DSPS protocols and provides helper classes to support the development of your own CodeLess™ and DSPS applications.

Import the codelesslib library module into your Android Studio project. You can use the SmartConsole Android application source code as reference.

For more information on how to use the library, refer to the following documentation pages.

  • Use {@link com.diasemi.codelesslib.CodelessScanner} to scan for Bluetooth devices.
  • Use {@link com.diasemi.codelesslib.CodelessManager} to connect and interact with a CodeLess™ or DSPS device.
  • {@link com.diasemi.codelesslib.CodelessCommands} contains helper methods for sending various AT commands to the peer device.
  • {@link com.diasemi.codelesslib.CodelessEvent} contains event classes for all possible events that can be generated by the library.
  • {@link com.diasemi.codelesslib.CodelessProfile} contains definitions and helper classes and methods.
  • See {@link com.diasemi.codelesslib.command.CodelessCommand} on how supported AT commands are implemented and how to add new commands.
  • Modify {@link com.diasemi.codelesslib.CodelessLibConfig} to configure various aspects of the library behavior.
  • Modify {@link com.diasemi.codelesslib.CodelessLibLog} to configure the log output produced by the library.

See also: CodeLess User Manual

Note: There is also a CodeLess™ library for iOS development, which is written in Objective-C. While class names may be slightly different and the API uses the Objective-C syntax, the iOS library behavior is similar to the Android one, except for some platform specific cases.
